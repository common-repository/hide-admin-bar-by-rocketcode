== Description ==

This plugin hides the frontend admin bar of your website by default, but allows you to select which user levels should still have access to it. With this plugin, you can easily choose which user roles should see the admin bar and hide it for all others.

== Installation ==

Upload the plugin files to the /wp-content/plugins/rocketcode_hab directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress
Go to the plugin settings page and select which user level you want to hide the admin bar from
== Usage ==

Once the plugin is installed and activated, the admin bar will be hidden from the front-end of your website, according to your settings. You can check the front-end of your website to see if the admin bar is hidden.
